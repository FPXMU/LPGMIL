# Learnable prototype-guided multiple instance learning for detecting tertiary lymphoid structures in multi-cancer whole-slide pathological images
![Fig_2](https://github.com/user-attachments/assets/315b8663-1aed-4f4a-83c1-3f5a8af11271)

(1) First, install the runtime environment according to the configuration file.
(2) Download the dataset from the TCGA dataset website at: https://portal.gdc.cancer.gov/
(3) For specific data processing and visualization, you can follow the complete process as outlined in CLAM (https://github.com/mahmoodlab/CLAM)
(4) The model algorithm is located in Model/LPGMIL.py. You can run it directly by executing python main.py --xxx -xxx
